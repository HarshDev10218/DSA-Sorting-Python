# Sorting Algorithms Collection

A collection of classic data sorting programs. This project demonstrates how five different fundamental algorithms take an unsorted list of numbers entered by a user and arrange them in ascending order.

## Purpose of the Project

The purpose of this project is to showcase and compare different methods for organizing data. While all five programs achieve the exact same final result—a perfectly ordered list—each one uses a completely different strategy and logic behind the scenes to get there.

---

## Included Algorithms

The collection includes the following five sorting methods:

1. **Bubble Sort**
   A simple approach that steps through the list, compares neighboring numbers, and swaps them if they are in the wrong order. It repeats this process until the entire list is sorted.

2. **Insertion Sort**
   This method builds the sorted list one number at a time. It takes each new number from the unsorted section and inserts it into its correct position among the numbers that have already been sorted.

3. **Selection Sort**
   This program scans the entire list to find the absolute smallest number, puts it at the beginning, and then repeats the process for the remaining numbers until everything is in order.

4. **Merge Sort**
   A highly efficient method that breaks the entire unsorted list down into individual single numbers. It then pairs them up and merges them back together piece-by-piece, ensuring they are perfectly ordered at every step.

5. **Quick Sort**
   This method selects a "pivot" number from the list and splits the remaining numbers into two groups: those smaller than the pivot and those larger. It then repeats this splitting process on the smaller groups until the whole list is organized.

---

## How It Works for the User

No matter which sorting program you choose to run, they all share a simple, interactive step-by-step workflow:

1. **Set the Size:** The program will first ask you to type in how many numbers you want to sort.
2. **Enter the Numbers:** You will be prompted to type in your numbers one by one.
3. **See the Result:** The program will immediately display your original unsorted list exactly as you typed it, followed by the newly organized, fully sorted list.
